# ucondofinancas
App para controle de fiananças. Desafio para o teste de Front end da UCondo

## Sobre o desenvolvimento
O app foi desenvolvido em React Native com Expo

## Para instalar
```bash
  npm install  
```
ou 
```bash
  yarn
``` 

## Para executar a aplicação no emulador
```bash
  npx expo start
```
